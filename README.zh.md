# 🛠️ 工具集合

一个基于 Nuxt.js 构建的实用工具集合，提供多种常用工具，支持中英文切换。

## ✨ 功能特性

- 🛠️ 多种实用工具集合
- 🌐 国际化支持 - 中英文切换
- 📱 响应式设计 - 适配各种设备尺寸
- 🚀 快速加载 - 基于 Nuxt.js 构建，性能优秀
- 🔄 持续更新 - 定期添加新工具

## 🛠️ 技术栈

- **框架**: Nuxt.js 4
- **前端**: Vue 3
- **样式**: Bulma CSS
- **图标**: Lucide Vue Next
- **国际化**: @nuxtjs/i18n
- **包管理器**: Bun
- **代码检查**: OXLint
- **代码格式化**: OXFmt

## 📦 安装

```bash
bun install
```

## 🚀 运行

### 开发服务器

启动开发服务器：

```bash
bun run dev
```

### 构建生产版本

```bash
bun run build
```

### 预览生产版本

```bash
bun run preview
```

## 🌍 国际化

项目支持中英文两种语言，可在页面右上角切换。
